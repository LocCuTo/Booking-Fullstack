# Booking-Fullstack
## Prerequisite
1. Install NodeJS 18.14.0 or 14.17.0
2. Type: 'npm install' in both nodejs & reactjs folder
3. Install xampp
4. Import database to xampp
5. Start xampp
6. Type: 'npm start' in both nodejs & reactjs folder
7. Enjoy
